# 增加应用内更新功能

## Goal

实现应用内自动更新功能，用户可以在应用内检查更新、下载APK并自动安装。

## Requirements

- 检查GitHub Releases是否有新版本
- 显示更新日志和版本信息
- 下载APK文件并显示进度
- 下载完成后自动安装
- 处理网络错误和下载失败
- 支持取消下载

## Acceptance Criteria

- [ ] 用户可以在设置页面检查更新
- [ ] 发现新版本时显示更新对话框
- [ ] 对话框显示版本号和更新日志
- [ ] 用户可以点击"下载更新"按钮
- [ ] 下载过程中显示进度条
- [ ] 下载完成后自动触发安装
- [ ] 用户可以取消下载
- [ ] 网络错误时显示错误提示
- [ ] 已是最新版本时显示提示

## Notes

- 需要Android安装权限
- 需要处理Android 10+的存储权限
- 需要使用dio库进行网络请求
- 需要使用open_file库打开APK文件
