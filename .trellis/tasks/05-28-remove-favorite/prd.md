# 移除我喜欢功能

## Goal

移除应用中的"我喜欢"功能，简化歌单管理。

## Requirements

- 移除歌曲详情页中的喜欢按钮
- 移除歌单页面中的"我喜欢"歌单显示
- 移除首页和最近播放页面中的"我喜欢"相关内容
- 移除通知设置中的喜欢操作选项
- 移除 PlaylistsService 中的 isFavorite 相关逻辑
- 保留普通歌单功能不受影响

## Acceptance Criteria

- [ ] 歌曲详情页不再显示喜欢按钮
- [ ] 歌单页面不再显示"我喜欢"歌单
- [ ] 首页和最近播放页面不再显示"我喜欢"内容
- [ ] 通知设置中不再显示喜欢操作选项
- [ ] 应用可以正常编译和运行
- [ ] 普通歌单功能不受影响

## Notes

- 需要谨慎处理数据库迁移，避免数据丢失
- 移除功能后需要测试其他功能是否正常
